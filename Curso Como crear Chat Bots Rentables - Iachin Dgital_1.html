¿Qué aprenderás en este curso?

Al terminar este curso estarás en condiciones de crear campañas de Chat Bots que logren resultados para tu negocio y los de tus clientes. Además te enseñaremos las estrategias que nos funcionaron a nosotros para conseguir nuestros clientes, así sabrás qué tipos de clientes buscar, cómo realizar la propuesta, cuáles palabras usar y cuáles evitar, y mucho más.

El curso está dividido en 6 módulos:

Módulo 1: Cómo crear tu Chat Bot

En este módulo cubriremos toda la parte técnica para que empieces a crear tu Chat Bot, y veremos todas las principales funcionalidades de la herramienta que utilizaremos.

Podrás acceder a clases paso a paso para que vayas siguiendo y armes tu primer Chat Bot.

Módulo 2: Perfeccionar el Contenido

La diferencia entre un Chat Bot exitoso y otro que enoja a sus usuarios suele ser su contenido.

En este módulo vamos a fondo con este tema para que puedas crear contenido que entretenga, informe y cree una relación con tu audiencia mientras al mismo tiempo logra los resultados que el negocio necesita.

Módulo 3: Consigue Suscriptores para tu Chat Bot

Existen varias herramientas para lograr que tus clientes potenciales entren y empiecen a usar tu Chat Bot. Analizamos cada una de ellas, y te mostramos cómo puedes usarlas según las características de cada campaña u objetivo que tengas.

Módulo 4: Cómo Conseguir Clientes que te paguen

Si proporcionas servicios de Marketing a otras empresas, este módulo es especialmente para ti.

Discutimos cuáles tipos de clientes son más propensos a invertir en una tecnología aún nueva como Chat Bots y cuáles características tienes que buscar para encontrar tu primer cliente.

Te contamos las estrategias que han funcionado para nosotros para buscar clientes, hacer demos, presentar la idea, y mucho más.

Módulo 5: Cómo Agregar Inteligencia con Integraciones

Cuando se integra el Chat Bot con otros sistemas se logra multiplicar el poder de esta herramienta de comunicación.

En este módulo te mostramos cómo puedes integrar tu Chat Bot con otras plataformas, y así crear soluciones mucho más complejas e interesantes.

Algunos ejemplos posibles: sistemas de turnos o reservas por Messenger, seguimiento de pedidos, recordatorio de carrito abandonado, descuentos inteligentes, integración con sistemas de email, y mucho más.

Poder crear soluciones complejas es lo que diferencia un buen desarrollador de Chat Bots de un mal diseñador o una plantilla predeterminada, y esta es una de las razones porque nuestros alumnos podrán cobrar más por sus Chat Bots.

Módulo 6: Bonus

Todos los días hay novedades en el mundo de Chat Bots. Dedicamos este último módulo para actualizarnos con las novedades, discutir otras herramientas disponibles para crear Chat Bots, y demás temas interesantes que no entren en el cuerpo principal del curso.
