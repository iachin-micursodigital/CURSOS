Description

Global Mapper es una aplicación GIS (Sistema de información geográfica) que ofrece acceso a una gran variedad de datos espaciales y proporciona el nivel de funcionalidad adecuado para satisfacer tanto, a usuarios experimentados en GIS como a usuarios iniciales.

Esta aplicación no solo permite ver, si no también editar o convertir, incluso imprimir varios mapas y conjuntos de datos vectoriales.

Características:
Herramienta de píxeles a puntos para crear una nube de puntos de alta densidad a partir de imágenes superpuestas
Una práctica barra de herramientas LiDAR para acceder fácilmente a las funciones clave de edición y análisis
Varias opciones de grillado para una generación más rápida de DSM o DTM
Acceso a archivos de nubes de puntos que contienen mil millones de puntos o más
Herramientas automáticas de clasificación de puntos que distinguen automáticamente las líneas de construcción, suelo, vegetación y energía y cables de utilidad sobre el suelo en capas no clasificadas
Característica de extracción de características para crear automáticamente huellas de edificios 3D, árboles y líneas eléctricas
Representación transversal con la herramienta de perfil de ruta de Global Mapper para ver y editar la nube de puntos en una perspectiva vertical
Digitalización 3D o extracción de características personalizadas utilizando la función de perfil perpendicular
Opciones avanzadas de filtrado para eliminar de manera eficiente los puntos erróneos o innecesarios
Detección y reclasificación personalizables de puntos de ruido
Comandos de scripts LiDAR para agilizar el flujo de trabajo
Colorización del punto de imágenes subyacentes o capa cuadriculada
Una herramienta de control de calidad LiDAR para corregir las elevaciones de nubes de puntos usando puntos de control de tierra
Múltiples opciones de visualización de nubes de puntos que incluyen clasificación, intensidad, elevación, RGB y densidad de puntos
Soporte para reportar estadísticas LiDAR
Soporte para importar y exportar formatos de nubes de puntos más comunes
Soporte para atributos de punto NIR (infrarrojo cercano) con la opción de crear una grilla NDVI o NDWI
Optimización de clasificación LiDAR para mejorar la velocidad de visualización y análisis.

ALGUNAS DE LAS TAREAS QUE PODEMOS HACER
===================================================================
– Cálculo de distancias y áreas
– Consultar elevaciones.
– Cálculo de volumen de corte y relleno.
– Delimitación de cuencas hidrográficas
– Generación de curvas de nivel desde una superficie

(Con esta opción podemos crear curvar de nivel y exportarlas en formato CAD que nos servirá para crear superficies en AutoCAD Civil 3D).
