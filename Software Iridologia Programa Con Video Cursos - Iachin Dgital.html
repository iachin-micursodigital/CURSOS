Description

Software destinado a la gestión de valoraciones iridológicas con soporte de manejo de imagen digitalizada que destaca por su facilidad de instalación y sencillo manejo e interpretación.
Debido a que esta dotado de un motor de bases de datos, está concebido para adaptarse a los diferentes entornos de la med alternativa y la naturopatía, además de proporcionarle flexibilidad con sus cualidades de funcionamiento para consulta iridológica cualitativa-cuantitativa.

Su módulo para configuración de especialidades se adapta para ingresar información al sistema para conformar cuadros básicos de med natural, fitoterapia, homeopatía, nutracéutica y muchas más.

Diseñado para trabajar en equipos locales y portátiles, incorpora un sistema de registro adecuado para la protección de la disponibilidad periódica de los datos de los usurarios que asisten a sesiones de valoración sin descuidar la confidencialidad de la información.

A nivel disciplinario el naturopata puede ajustar la aplicación de las más variadas topologías, como la configuración tipo Americana, Alemana e integral.
Ficha técnica de descripción operativa:

* Ficha del paciente completa, con antecedentes y curso pre-clínico.
* El usuario crea cuadros básicos de terapias alternativas según su especialidad.
* El usurario realiza valoraciones cualitativas y valoraciones cuantitativas basadas en datos estadísticos con infografías de sistemas corporales.
* Sistema configurable para anotar comentarios de cualquier tipo al momento de la exploración iridiana.
* Impresión de recetas, con datos distintos para cada usuario.
* Histórico de visitas configurable con soporte periodico para copias de seguridad.
* Utilidad de exportación de fuentes Twain para captura de imagen digitalizada para formatos diversos.
* Cumple con los requisitos técnicos exigidos por iridoscopios CMOS y CCD.
* Compatible Windows 7 32 bits
* Adaptado a portátiles tipo Laptop.
* Sólo necesita instalación, copie los ficheros y ya esta listo para funcionar.
* Utilidades para configurar los parámetros de funcionamiento del programa y adaptaciones para calibración de iridoscopios o dispositivos de captura de imagen a formato digital.
