Descripción del producto

Programa adaptado para consultorios con no más de cinco profesionales, tiene un agenda muy facil de utilizar, no maneja bonos ni cobros. Puede llevar un registro de las visitas de los pacientes, incluso cargar imágenes, radiografías o fotografías.

Características del programa de historias clínicas

• Agenda de Reservas donde puede anotar el horario de la persona y el profesional con quien se atiende, con la posibilidad de imprimir los turnos del día.

• Puede llevar facilmente la historia clínica del paciente, cargar fotografías y consultar las notaciones de visitas anteriores.

• Ficha del paciente, donde podrá cargar todos los datos referidos al mismo.

• Seguridad de los datos implementada a través de permisos de usuario del programa.

En caso de que sea necesario, también es posible crear diferentes usuarios para trabajar con el programa.

Para mayor seguridad, el programa realiza una copia de la base de datos al cerrarse; copia que puede ser restaurada en caso de problemas
