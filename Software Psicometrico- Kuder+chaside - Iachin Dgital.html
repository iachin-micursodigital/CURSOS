Description

Software para la aplicación automatizada del test KUDER de Intereses Personales, test Chaside y test de Intereses Vocacionales, lo cual convierte este producto en una batería de test dirigida a la evaluación de intereses personales y habilidades operativas en tareas de Orientación Vocacional. En la versión de este producto se posibilita la evaluación sistemática de las habilidades en 7 áreas de interés. Con este software los resultados se arrojan de manera automática, lo cual comprende los porcentajes y descripciones verbales del perfil correspondiente a KUDER, CHASIDE e Intereses Vocacionales, reduciendo hasta un 80% el tiempo neto de aplicación.

Las áreas que evalúa el software son:

1.- Profesiones del área económica-administrativa: Administración de empresas, secretariado ejecutivo, contabilidad, ingeniería industrial, etc.
2.- Profesiones del área de humanidades y ciencias sociales: Sociología, politología, psicología, antropología, etc.
3.- Profesiones del área artes: Licenciatura en música, pintura, filosofía, etc.
4.- Profesiones del área de ciencias médicas y de la salud: Medicina, optometría, cirujano dentista, etc.
5.- Profesiones del área de ingeniería y computación: Matemáticas aplicadas a la computación, sistemas informáticos, programación, etc.
6.- Profesiones del área de defensa y seguridad.
7.- Profesiones del área de ciencias exactas y agrarias: Física, QFB, Biología, ciencias del mar, etc.

La presentación de los test es la siguiente:

a) KUDER: Software de aplicación.
b) Chaside: Hoja de Excel.
c) Intereses Vocacionales: Software de aplicación.

Las tres escalas están integradas a un sólo software de Sistemas Psicológicos.
