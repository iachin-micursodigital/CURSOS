Curso de WordPress avanzado

Curso de WordPress avanzado en el que vamos a ir un paso más allá, para aprovechar al máximo todo lo que podemos llegar a hacer con WordPress.

No en vano WordPress es el CMS más usado en todo el mundo. Es muy fácil de usar, y su gran versatilidad te permite hacer todo lo que se te pase por la cabeza.

Si quieres hacer este curso, te recomiendo que primero hagas el de Curso de WordPress básico para conseguir un buen nivel de usuario, y luego el de Curso de WordPress Intermedio para tener un nivel de implementador iniciado. Ambos cursos están en nuestra página

A lo largo de 10 clases vamos a ver cómo modificar WordPress para adaptarlo a nuestras necesidades o a las de nuestros clientes o proyectos, en el caso que estemos interesados a tener un nivel de implementador avanzado, e incluso de programador básico.

Todos los capítulos de este curso WordPress Avanzado:
  • Curso de WordPress Avanzado #1: Modificar WordPress
  • Curso de WordPress Avanzado #2: Custom Post Types
  • Curso de WordPress Avanzado #3: Taxonomías
  • Curso de WordPress Avanzado #4: Ubicaciones de menús
  • Curso de WordPress Avanzado #5: Widget Areas
  • Curso de WordPress Avanzado #6: Menús en la toolbar
  • Curso de WordPress Avanzado #7: Métodos de contacto del usuario
  • Curso de WordPress Avanzado #8: Crear shortcodes
  • Curso de WordPress Avanzado #9: Scripts y estilos
  • Curso de WordPress Avanzado #10: Hooks
