Description

Microsoft Excel es un software de uso indispensable en la mayoría de profesiones y puestos laborales en todo el mundo. Es una importante herramienta que, sabiendo usarla correctamente, permite reducir considerablemente el tiempo de ejecución de diversas tareas como creación de informes, administración de bases de datos o creación de tablas dinámicas para el manejo de grandes cantidades de información.

¿Qué voy a aprender?
En este curso, aprenderás todas las funcionalidades de las hojas de cálculo, desde generar una estructura adecuada, hasta la aplicación y modificación de los formatos. Además, aprenderás a crear fórmulas aritméticas y estadísticas, para manejarlas de manera sencilla. Finalmente, serás capaz de ordenar y filtrar información de una base de datos.
