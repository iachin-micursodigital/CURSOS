Description

Aprende a planificar en modo profesional tus proyectos.
En el curso de Project 2016, aprenderás el uso de herramientas esenciales para crear y manejar proyectos.
Explorarás las funciones principales, desde tareas esenciales, como crear tareas y vincularlas, cómo modificar las relaciones existentes hasta cómo crear un plan de proyecto completo, incluyendo la creación de distintos tipos de recursos.
Verás también temas imprescindibles, como crear un calendario laboral y asignarlo al proyecto, ver la ruta crítica del proyecto y las tareas críticas que pueden demorar la fecha de finalización del proyecto. Aprende a sacar partido a Project 2016 para Windows.
