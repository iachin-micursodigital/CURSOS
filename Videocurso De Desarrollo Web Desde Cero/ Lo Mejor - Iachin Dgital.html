Description

En más de 45 horas de contenido, este extenso curso sobre programación te enseñará todo lo que necesitas saber sobre cómo convertirte en un desarrollador web. “Desarrollo Web” es el término utilizado para definir todo el trabajo involucrado en la creación de una página o aplicación web. Desarrollo web puede ir desde la creación de una página web simple y estática hasta las aplicaciones web más complejas, como por ejemplo: Facebook, Ebay, o Google Maps.

El curso empieza enseñando cómo funciona , con una introducción a HTML, XHTML, y CSS. Aprenderás cómo crear texto, listas, enlaces, metadatos, imágenes, tablas, cell-spacing y más. En la segunda Sección pasarás a JavaScript donde aprenderás cómo utilizar scripts externos, comentarios, variables, loops, arrays y alertas, así como otras funcionalidades de JavaScript. La tercera Sección trata sobre PHP y MySQL: strings, números, constantes, matrices, operadores, sesiones, paginación, bases de datos, cookies, etc (como puedes ver, el curso es EXTENSO – para más detalles mira la lista de abajo). Las cuarta y quinta Secciones tratan sobre XML y JSON, pasando después para AJAX. La sétima Sección trata sobre jQuery: efectos, eventos, funciones de devolución de llamadas, menús desplegables, autocomplete y cajas de diálogo. La octava Sección pasa para HTML5 y CSS3: artículos, elementos, jerarquías, pseudo clases y non-pseudo clases, animaciones, transformaciones y más. Este curso te dará un conocimiento fundamental sobre desarrollo web –

“¡Desarrollo Web Desde Cero!” te dará una nueva perspectiva sobre cómo funciona. Después de haber completado el curso, tendrás la amplitud de conocimientos que te permitirán ser capaz de crear tus propias páginas y aplicaciones web con la tecnología más reciente.
