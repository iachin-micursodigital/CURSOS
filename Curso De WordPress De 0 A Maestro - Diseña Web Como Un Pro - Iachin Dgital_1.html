Primer Proyecto: Sitio para un Colegio

El objetivo de este proyecto es descargar y usar una plantilla gratuita para WordPress y personalizarla para darle el look que necesitamos y crear un proyecto web para un colegio totalmente funcional y profesional.

Segundo Proyecto: Creando una Plantilla desde Cero

Este proyecto es un poco más orientado a los desarrolladores, pero si no lo eres no importa, el capítulo es fácil de seguir y se necesitan unos conocimientos básicos de html, css, php, los cuales puedes adquirir en mi canal en youtube “render2web”. En este capítulo vamos a crear una plantilla para WordPress partiendo desde una maquetación hecha en Bootstrap y te recomiendo que lo veas, te ayudará a entender mucho acerca de la estructura de las plantillas.

Tercer Proyecto: Creando un Plugin desde Cero

Este proyecto esta orientado a la creación de un plugin que nos permita crear una galería de videos de youtube, necesitarás unos conocimiento básicos de html, css y php los cuales puedes adquirir en mi canal en youtube “render2web”. Si no eres desarrollador ni te interesa conocer sobre programación para Worpdress puedes saltar esta clase y continuar con el siguiente proyecto, pero te recomiendo encarecidamente que lo hagas pues te ayudará a entender como funcionan los plugins en Worpdress y te servirá cuando necesites editar alguno de los existentes.

Cuarto Proyecto: Creando una Tienda en Línea con Woocommerce

Este es el proyecto más grande del curso en donde construiremos paso a paso y desde cero una tienda en línea totalmente funcional y de aspecto profesional usando WordPress y Woocommerce.
