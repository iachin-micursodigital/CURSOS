PAQUETE: IPV, LIFO, BFQ. Evaluación básica para la selección de personal.

Software que integra tres automatizaciones en excel de tres pruebas psicométricas típicamente aplicadas en procesos de selección y evaluación de personal; IPV, LIFO, BFQ haciendo de este producto una batería de evaluación dirigida a RRHH óptima para evaluar tanto operativos como bandos medios. Con este software los resultados se arrojan de manera automática, incluyendo todas las tablas de estadísticos, porcentajes así como gráficos correspondientes al Inventario Para Vendedores, LIFO que está dirigida a bandos medios y gerenciales, así como Big Five que es un test de personalidad, reduciendo hasta un 80% el tiempo neto de aplicación e interpretación de los test.

Inventario Para Vendedores
IPV se trata del test psicométrico estándar para evaluar potencial competencia de venta en individuos mayores de edad. El test evalúa el factor general de disposición para las ventas así como 11 factores que en confluencia otorga el perfil potencial de ventas.

Los factores que evalúa el test son:

a) Disposición General para la Venta
b) Receptividad
c) Agresividad
d) Comprensión
e) Adaptabilidad
f) Control de sí mismo
g) Tolerancia a la Frustración
h) Combatividad
I) Dominancia
j) Seguridad
k) Actividad
l) Sociabilidad

:::

LIFO
Test dirigido a la medición del Perfil de Fuerzas Productivas y Defensivas, comúnmente dirigido a la aplicación en bandos medios (supervisores y coordinadores), así como directivos. El test aborda 8 condicionantes de la conducta en los evaluados así como un parámetro de confiabilidad de indicadores.

Las condicionantes que mide el test son:

a) Cond. Favorables Colabora
b) Cond. Favorables Controla
c) Cond. Favorables Conserva
d) Cond. Favorables Adapta
e) Cond. Desfavorables Colabora
f) Cond. Desfavorables Controla
g) Cond. Desfavorables Mantiene
h) Cond. Desfavorables Adapta

:::

Big Five
Se trata de una prueba psicométrica dirigida a evaluar dimensiones fundamentales de la personalidad, 5 básicas y 5 extensas, así como medición de grados de confiabilidad. Su aplicación se encuentra optimizada para personas de 15 años en adelante, con estandarización en población mexicana. Las dimensiones con las que opera la prueba son:

a) Dinamismo
b) Dominancia
c) Cooperación
d) Cordialidad
e) Escrupulosidad
f) Perseverancia
g) Control de Emociones
h) Control de Impulsos
I) Apertura a la cultura
j) Apertura a la experiencia
