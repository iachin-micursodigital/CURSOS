multiecuscan es un software de diagnóstico de vehículos con funciones avanzadas. Le permite realizar diversas tareas de diagnóstico en los vehículos / módulos admitidos. Para usar el software necesita una interfaz. Las siguientes interfaces son compatibles actualmente con el software: KL (también conocido como VagCom 409), ELM327 (1.3 o más reciente), OBDKey 1.40, OBDLink, ELM Scan 5, CANtieCAR. Las interfaces Bluetooth OBDKey, ELM 327 y OBDLink son totalmente compatibles, pero no se recomiendan para funciones especiales (como alineación PROXI, programación de control remoto, codificación IMA, etc.). CANtieCAR es una interfaz profesional multiprotocolo con capacidades de multiplexación.
