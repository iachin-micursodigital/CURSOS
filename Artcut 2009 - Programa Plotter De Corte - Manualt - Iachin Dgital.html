CARACTERÍSTICAS DEL SOFTWARE
El software ARTCUT ofrece una solución para la necesidad de crear signos signos, desde un plotter corte de vinilo básico hasta la impresión avanzada en gran formato. Software que contiene herramientas para la creación y edición de fácil uso para que usted pueda completar sus funciones de diseño. Incluido en Artcut, programa de diseño y corte, las características tales como escaneo, escala de grises y trazado de imagen en color, manipulación de texto completo, archivos de importación / exportación, múltiples deshacer y rehacer. También incluye controladores directos para los populares cortadores de la industria.

Algunas funciones:

Editar: Copiar / Pegar, Rehacer / Deshacer, Duplicación, Grupo, Reemplazar Nodo editar, medir …
Texto: Truetype-Font de apoyo, la alineación del texto a lo largo de arte y curva, Transformación, atribuir editar, convertir a la curva …
Gráfico: Rectángulo, Elipse, Polígono, Línea, Arco, Estrella, Bezier, dibujo de la mano ….
Manipulación: Mover, Cambiar tamaño, rotación, espejo, Transformación, Smooth / Sharp, Inline / Esquema, Split …
Ver: Barra de estado, Biblioteca, barra de herramientas, Cuadrícula, Ratio, línea de base …
Funciones adicionales: Multi-Ventana, Multi-páginas, tamaño de página, línea de corte, línea de Trim, Bloquear / Desbloquear, Coordinador cambiar …
Importación del archivo: Formato de datos: eps, HP-GL, AI, DXF, texto, imagen, Escanear …
