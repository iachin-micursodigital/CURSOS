Description

Aprende lo esencial y a usar el software de diseño y modelado paramétrico de Autodesk Revit 2017

Los proyectos arquitectónicos resultan cada vez más complejos y requieren de mayor flexibilidad, pero Autodesk Revit ha logrado adaptarse haciendo posible la creación de modelos de información para la construcción (BIM). En este curso aprenderás a trabajar con las herramientas de modelado, a editar elementos paramétricos para ajustar tu modelo rápidamente, aprenderás a trabajar con vistas y obtener información rápida del modelo, que hará más sencilla su integración con otras áreas como las ingenierías, relacionadas con la creación de un proyecto ejecutivo.
