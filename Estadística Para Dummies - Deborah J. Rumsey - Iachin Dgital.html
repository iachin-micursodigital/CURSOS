ESTADÍSTICA PARA DUMMIES
Autor(es): DEBORAH J. RUMSEY

Sinopsis:

Esperanza de vida, tasa de siniestralidad, tablas, promedios… cada día recibimos centenares de datos estadísticos, a menudo sin tener las herramientas suficientes para analizarlos.

El objetivo de este libro es ayudar al lector a desarrollar y fortalecer sus habilidades de análisis para que pueda evaluar con solvencia datos de este tipo y tomar decisiones sensatas en, por ejemplo, la contratación de seguros. Y el aprendizaje será Para Dummies, es decir, sin dolor ni sufrimiento.
