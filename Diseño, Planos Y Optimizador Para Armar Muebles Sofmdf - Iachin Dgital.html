Description

Software multimedia ideal para quienes quieran diseñar muebles en melamina, en su versión 1.8 se incluye un diseño adicional para bajo mesada estilo americano!!! además de diseños varios que le permitirán por ejemplo crear su propia Biblioteca o armar un nuevo ropero para su cuarto o si prefiere podría crear un armario estilo moderno para su casa u Oficina utilizando las placas de melamina, además podrá remodelar su cocina diseñando su propia bajo mesada y alacena con trazos modernos al mejor estilo americano!! Podrá crear también mesas de escritorio o muebles para PCs y mucho más… todo esto utilizando las medidas y planos que se ofrecen navegando por el menú del programa.

Mejoras:

En su nueva versión encontrara un menú visualmente mejorado, intuitivo y fácil de usar para que el usuario pueda ver los planos con las medidas, trazados y el explicativo de cómo desarrollar el armado de forma sencilla y clara. También permite impresión en papel a A4.
Además contiene un optimizador de láminas para no desperdiciar espacios y hacer rendir al máximo cada placa de melamina.
El Software multimedia funciona bajo sistema operativo Windows en todas sus versiones (XP, Windwos 7, 8 y 10).
