Description

Componer Canciones. Curso Básico

Adquiere los conocimientos que te ayudarán a componer mejor tus canciones.

Lo que aprenderás

Con este curso aprenderás una serie de herramientas que te ayudarán mucho a la hora de componer tus canciones.

Requisitos

No requiere que tengas conocimientos musicales o literarios previos para iniciar este curso. Solo necesitas tu creatividad y pasión por la música. Si te puedes ayudar de un instrumento como un teclado o una guitarra, mejor.

Descripción

Con este curso adquirirás Herramientas para Componer tus Canciones.Ampliarás tus conocimientos musicales para que puedas aplicarlos de forma fácil y rápida

El lenguaje y los conocimientos que adquirirás son de un nivel básico/intermedio, de sencilla comprensión e inmediata asimilación para ponerlos en práctica ya.

El curso incorpora 21 lecciones en video acompañadas de un apartado de ejercicios para cada una de las clases y otro de soluciones para autoevaluarte

El curso se estructura dentro de cuatro bloques independientes

* La simbología musical

* La tonalidad y los acordes

* La melodía

* La Letra

Tanto si tienes algo de conocimientos musicales o literarios como si no los tienes. Este curso, una vez finalizado, te abrirá un abanico de posibilidades que antes no habrías imaginado a la hora de crear tus canciones

¿Para quién es este curso?

Este curso está destinado a ti que eres una persona creativa y expresa tu creatividad escribiendo canciones
