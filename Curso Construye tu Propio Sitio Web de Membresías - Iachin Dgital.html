Todo esto lo vamos a hacer empezando de cero y llevándote de la mano… paso a paso

Módulo 1 Introducción y tipos de membresías

Te enseñaremos con todo detalle cómo funciona un sitio web de membresías y también te daremos algunas ideas sobre cómo puedes cobrar por una membresía o suscripción por tus servicios o productos

Módulo 2 Herramientas y servicios necesarios 

Te diremos qué servicios y herramientas son necesarias para construir una plataforma de membresías profesional y donde encontrarlas.

Compartiremos contigo las herramientas básicas, necesarias para lanzar tu negocio pero también si quieres llevar tu negocio al siguiente nivel te diremos cuales son las herramientas de marketing más avanzadas que existen en el mercado. Estas son las herramientas que nosotros mismos utilizamos y que han hecho que nuestro negocio crezca mes tras mes.

Módulo 3 Hosting y Dominio

3.1 Cómo contratar un dominio y un hosting 

3.2 Cómo acceder a WordPress 

3.3 Qué es y cómo funciona WordPress 

3.4 Cómo instalar un tema para tu plataforma de WordPress 

Te llevaremos de la mano en todo el proceso que tienes que seguir para conseguir un dominio, contratar un hosting e instalar la plataforma de WordPress a el para así poder empezar a construir tu plataforma de membresías

Módulo 4 Domina WordPress 

Este módulo en sí mismo es un curso totalmente completo sobre WordPress, una de las plataformas más poderosas para crear sitios web.

Encontrarás todo lo que necesitas saber para dominar esta plataforma

Módulo 5 La estructura de tu sitio web 

En este módulo te enseñaremos a organizar todo el contenido de tu sitio web perfectamente en un mapa para que puedas encontrar fácilmente todas las páginas que vayas creando e iremos construyendo paso a paso tu sitio web.

Te enseñaré cómo abrir una cuenta en Amazon Web Services, para que utilices su servicios S3 y CloudFront. Este servicio es crucial, para que tus videos en Streaming corran rápidamente.

También descubrirás el VideoPlayer profesional que nosotros mismos utilizamos y cómo integrarlo con Amazon Web Services. Este VideoPlayer puede convertir tus videos en verdaderas máquinas de vender.

Módulo 6 Convierte tu sitio de WordPress en una plataforma de membresías

Este módulo lo dedicaremos a configurar tu plataforma de membresías para que puedas cobrar de manera recurrente y automática a tus clientes por tener acceso a determinado contenido de tu sitio web.
